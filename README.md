###Tasks

#Description:
Todo list program.

#user must return to main menu after each task.

##Main functionality:

#ADD a task:
task:
*title,
*description,
*deadline,
*status (Todo, In progress, finished);

#Display a list of all tasks:
3 ways (submenu):
*By alphabetical order,
*By deadline,
*deadline first that are urgent (less than 3 days left),

#Modify a task:
*modify the description,
*modify the status,
*modify the deadline,

#Delete a task:
by id,

#Search for a task:
by id,
by title,

#Display statistics:
3 ways (submenu):
*total number of tasks,
*number of complete and incomplete tasks,
*days left until deadline of each task,
